## House Price Prediction — ML Project
### Описание проекта

В данном проекте реализована модель машинного обучения для прогнозирования цены дома на основе его характеристик. Работа включает полный цикл анализа данных: исследовательский анализ (EDA), проверку взаимосвязей признаков, построение и интерпретацию линейных моделей регрессии.

## Данные

Используется датасет с Kaggle (https://www.kaggle.com/datasets/prokshitha/home-value-insights) из 1000 объектов со следующими признаками:

- площадь дома,

- количество спален и ванных,

- год постройки,

- размер участка,

- наличие гаража,

- качество района.

Целевая переменная — **House_Price**.

### 🔍 Анализ данных

- Проведён EDA с визуализацией распределений и корреляций

- Выявлено, что площадь дома является основным фактором, влияющим на цену

- Проверена мультиколлинеарность признаков с помощью VIF

### Модели

- Linear Regression

- Regularized Linear Model (Ridge)

## Результаты

Модели показали высокое качество прогнозирования:

- R² ≈ 0.99

- MAE ≈ 18 000

- RMSE ≈ 22 000

Регуляризация не дала значимого прироста качества, что говорит о корректной структуре данных и устойчивости модели.

### Интерпретация

Модель интерпретируема: влияние каждого признака на цену дома может быть объяснено количественно, что важно для бизнес-задач и принятия решений.
